## üîî Notification Settings

### Basic Settings

**Enable Notifications**
- Master toggle for all notifications
- Turn off to stop all alerts

**Play Alarm Sound**
- Audible alert when tasks are due
- Requires notifications to be enabled

**Loop Alarm Sound**
- Continuously repeats until dismissed
- Great for ensuring you don't miss deadlines

### Desktop Notifications

**What are Desktop Notifications?**
Desktop notifications appear even when:
- The app is minimized
- You're on a different browser tab
- You're using another application

**How to Enable:**
1. Check "Desktop notifications"
2. Click "Request Permission"
3. Allow in your browser prompt

**Permission Status:**
- ‚úÖ Granted: You're all set!
- ‚ö†Ô∏è Not yet requested: Click the button to enable
- üö´ Blocked: Change in browser settings

**Unblocking Desktop Notifications:**
- **Chrome/Edge**: Click the lock icon ‚Üí Site settings ‚Üí Notifications
- **Firefox**: Click the shield icon ‚Üí Permissions ‚Üí Notifications
- **Safari**: Safari menu ‚Üí Settings ‚Üí Websites ‚Üí Notifications

### Notification Timing

**Notify before due time (minutes)**
- Set how early you want to be alerted
- Example: "5" means alert 5 minutes before deadline
- Set to "0" for alerts at exact due time

**How it works:**
If a task is due at 10:30 AM and you set 5 minutes:
- You'll get notified at 10:25 AM
- Gives you time to prepare!

### Test Alarm
Click "Test alarm sound" to:
- Verify your speakers/headphones work
- Hear what the alarm sounds like
- Test plays for 3 seconds

## üÜò Troubleshooting

**Problem: Settings don't save**
- Solution: Check if browser allows localStorage
- Try: Clear browser cache and retry

**Problem: Desktop notifications not working**
- Check: Browser permission status
- Try: Click "Request Permission" again
- Verify: System notifications are enabled

**Problem: Alarm sound not playing**
- Check: Browser autoplay policy
- Try: Click "Test alarm sound"
- Verify: System volume is not muted

**Problem: Theme not changing**
- Note: Only Retro Dark is currently implemented
- Other themes: Coming in future updates

**Problem: Compact mode doesn't work**
- Note: Visual changes coming in future update
- Current: Preference is saved for future use

## üîí Privacy & Data

**What's stored locally:**
- Display name
- Email address
- Theme preference
- Animation settings
- Notification preferences
- Session timestamps

**What's NOT stored:**
- Your password
- Task data (stored separately)
- Payment information
- Browsing history

**Data location:**
- All settings stored in browser's localStorage
- Data never leaves your device
- Clearing browser data removes all settings
